# Compound Interest Calculator

A small interactive React app that visualizes compound interest growth with an animated canvas chart.

Adjust the initial amount, monthly contribution, annual return rate, and investment period with sliders, and watch the chart update in real time. It also shows total balance, total contributions, total interest earned, and the Rule of 72 (how many years it takes for your money to double).

## Built With

- React (Create React App)
- HTML5 Canvas
- Google Fonts (DM Sans / DM Mono)

## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/keymil0/compound-interest.git
   ```
2. Go into the project folder
   ```sh
   cd compound-interest/compound-interest
   ```
3. Install dependencies
   ```sh
   npm install
   ```
4. Run it
   ```sh
   npm start
   ```
5. Open [http://localhost:3000](http://localhost:3000)
